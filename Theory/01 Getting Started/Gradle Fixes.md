## Stop All Running Gradle Daemons  
```bash
# cmd
./android/gradlew.bat --stop
```  
```bash
# bash
./android/gradlew --stop
```  

## Clean the Gradle Build Cache  
```bash
# cmd
./android/gradlew.bat clean
```  
```bash
# bash
./android//gradlew clean
```  

## For other errors  
```bash
npx react-native doctor
```
